README - Java Exercise

Requirements:
- Java installed (javac and java commands available).
- Run commands from the folder where the .java file is located.

------------------------------
EXERCISE - Recursive directory listing saved to a dated TXT file
------------------------------
Compile:
javac DirectoryList.java

Run:
java DirectoryList folder_name

Example:
java DirectoryList examples

This will create a file named like "listing_2025-07-13_15-30.txt" with the directory listing.
